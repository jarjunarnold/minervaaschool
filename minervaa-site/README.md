# Minervaa Vidhya Mandhir Website Starter

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this folder to GitHub.
2. Import the repo in Vercel.
3. Add your custom domain `minervaaschool.com` in Vercel project settings.
4. Update DNS records in GoDaddy as shown by Vercel.

## Important content updates before launch

- Replace `+91 XXXXX XXXXX` with the real phone number.
- Replace `info@minervaaschool.com` if needed.
- Add the exact school address.
- Replace the founder photo placeholder with a real image.
- Connect the enquiry form to Formspree, Resend, or a server action.
